## Getting Started

To get started, follow these steps:

1. **Clone the Repository**:

   ```bash
   git clone git@github.com:iiishi9779/avas-demo.git
   ```

2. **Install Dependencies**:

   ```bash
   cd avas-demo
   bun install
   ```


3. **Run the Development Server**:

   ```bash
   bun --bun run dev
   ```
